### Local environment
- Clone the repo locally `git clone https://github.com/unitaryfund/unitaryhackdev.git`
- Navigate to root folder `cd untitaryhack2022`
- Install the goods `npm install`
- Run it `npm start`
- You should now be able to see everything running on localhost:8080
- Make a Pull Request to add your changes to github. Once approved, an auto-deploy should be triggered ğŸ‰
